🔧 Admin Setup

Use this screen to configure the app password and embed your GitHub token. You do this once, then share the updated HTML file with your team. Staff will only ever see the lock screen.

Step 1 — Enter your GitHub details

From GitHub → Settings → Developer settings → Personal access tokens. Needs repo scope.

Step 2 — Set the access password

Share this with your team. It encrypts your GitHub token — without the right password, the token cannot be read.

Step 3 — Generate encrypted blob

Click the button below. This turns your token into an encrypted string that is safe to store in the HTML file — it's meaningless without the password.

Encrypted blob will appear here…

Step 4 — Embed the blob in the HTML file

  1. Copy the blob above
  2. Open the HTML file in a text editor (Notepad, TextEdit, VS Code, etc.)
  3. Find the line that says const ENCRYPTED_BLOB = '';
  4. Paste your blob between the single quotes so it looks like const ENCRYPTED_BLOB = 'abc123…';
  5. Save the file and share it with your team
The admin setup screen is always accessible from the lock screen link below — you'll need the admin password to get in. Keep that password separate from the staff password.
